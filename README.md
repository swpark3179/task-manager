# 작업 관리자 (Task Manager)

작업 관리자는 매일의 업무와 개인 할일을 효율적으로 관리하기 위한 생산성 도구입니다. 이 프로그램은 할일 목록 관리, 달력 보기, 히스토리 조회 등 다양한 기능을 제공하여 체계적인 일정 관리를 돕습니다.

## 주요 기능

### 1. 오늘의 할일 관리
- **할일 추가**: 화면 상단의 입력창에 할일을 입력하고 Enter를 눌러 손쉽게 추가할 수 있습니다.
- **하위 할일**: 각 할일 아래에 더 작은 단위의 세부 할일을 추가하여 작업을 쪼개어 관리할 수 있습니다.
- **상태 관리**: 할일을 '미수행', '진행 중', '완료', '폐기' 상태로 관리할 수 있습니다. 체크박스를 클릭하여 상태를 변경하세요.
- **상세 내용 작성**: 각 할일의 세부 내용을 마크다운(Markdown) 형식으로 작성할 수 있습니다.
- **수행 내역 기록**: 할일에 대한 진행 상황을 시간별/일별로 기록할 수 있습니다.
- **자동 이월**: 전날 완료하지 못한 할일은 다음 날 앱 실행 시 자동으로 오늘 날짜로 이월되어 누락을 방지합니다.

### 2. 달력 뷰
- 월간 달력 형태로 매일의 작업 현황을 한눈에 파악할 수 있습니다.
- 날짜별로 완료, 진행 중, 대기 상태의 작업 개수를 색상 점으로 표시합니다.
- 달력의 날짜를 클릭하면 해당 날짜의 작업 내역(히스토리)으로 바로 이동합니다.

### 3. 히스토리
- 과거 특정 날짜의 작업 내역을 조회하고 수정할 수 있습니다.
- 상단의 날짜 이동 버튼을 사용하여 이전/다음 날짜로 쉽게 이동할 수 있습니다.

### 4. 데이터 내보내기
- 설정 화면에서 모든 작업과 수행 내역을 마크다운(.md) 파일 형태로 내보낼 수 있습니다.
- 오프라인 백업이나 다른 도구와의 연동 시 유용하게 사용할 수 있습니다.

### 5. 오프라인 & 동기화 (클라우드)
- 데이터는 Supabase를 통해 안전하게 클라우드에 저장되며, 실시간으로 동기화됩니다.
- 화면 우측 상단의 동기화 상태 표시기를 통해 현재 데이터 저장 상태(동기화 중/완료)를 확인할 수 있습니다.

## 사용 방법 (팁)

1. **상위-하위 작업 구조 활용하기**:
   큰 규모의 작업은 부모 할일로 등록하고, 세부 실행 단계는 하위 할일로 등록하세요. 하위 할일의 완료 비율에 따라 부모 할일의 진행률이 자동으로 계산되어 표시됩니다.
2. **수행 내역 꼼꼼히 기록하기**:
   작업이 길어질 경우 '수행내용' 탭에 진행 과정을 기록해두면, 나중에 작업 히스토리를 파악할 때 큰 도움이 됩니다.
3. **더블 클릭 수정**:
   할일 목록에서 제목을 더블 클릭하면 빠르게 제목을 수정할 수 있습니다.

## 설정 및 기타

- 설정 메뉴에서 계정 로그아웃, 데이터 내보내기 및 네트워크 환경에 맞춘 프록시(Proxy) 설정을 변경할 수 있습니다.
