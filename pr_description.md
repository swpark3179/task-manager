🎯 What
일정(Schedule) 등록 모달의 기능을 개선하고 버그를 수정했습니다.

- 데이터베이스에 public.schedules 테이블이 존재하지 않던 문제 해결 (`20240422000000_add_schedules.sql` 마이그레이션 적용)
- 일정 수정 모달에서 등록된 일정의 종료일을 변경할 수 있도록 활성화 (시작일은 변경 불가, 신규 생성 시에만 시작/종료일 모두 변경 가능)
- 기존 일정을 삭제하는 기능 추가
- 달력 뷰에서 일정을 클릭했을 때 수정 모달이 열리도록 변경

📊 Coverage
- `bun test` 명령어로 테스트 코드 실행 및 통과 확인 완료.

✨ Result
- 사용자가 일정을 보다 원활하게 등록, 수정, 삭제할 수 있게 되었습니다.
- 달력 페이지에서 일정을 직접 관리할 수 있어 사용자 경험이 향상되었습니다.
